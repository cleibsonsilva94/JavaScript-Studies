# 📌 JavaScript + Vue.js — Star Wars Characters

Este projeto tem como objetivo demonstrar conceitos fundamentais de desenvolvimento front-end utilizando **JavaScript moderno** e o framework **Vue.js**, aplicados na construção de uma interface dinâmica simples.

---

## 🎯 Objetivo do Projeto

- Compreender como frameworks JavaScript simplificam o HTML
- Comparar JavaScript puro vs Vue.js
- Trabalhar com renderização dinâmica de listas
- Aplicar conceitos de reatividade e data binding

---

## 🧠 Conceitos Aprendidos

### ✅ JavaScript
- Criação e manipulação de arrays e objetos
- Uso de constantes para modelar dados
- Funções para lidar com eventos do usuário
- Organização do código por responsabilidade

### ✅ Vue.js
- Criação de uma instância Vue
- Associação ao DOM via `el`
- Uso do objeto `data` como estado da aplicação
- Definição de comportamentos com `methods`

---

## 🔧 Técnicas Utilizadas

- **Renderização de listas** com `v-for`
- **Interpolação de dados** com `{{ }}`
- **Data Binding de atributos** com `v-bind` (`:src`)
- **Eventos** com `@click`
- **Reatividade automática** do DOM
- Separação entre dados, lógica e estrutura HTML

---

## 🧩 JavaScript moderno simplificando o HTML

Em aplicações modernas, o HTML deixa de ser apenas estático e passa a ser **declarativo**.

Com Vue.js:
- Não é necessário manipular o DOM manualmente (`querySelector`, `innerHTML`)
- O HTML descreve **o que será exibido**
- O JavaScript controla **os dados e o comportamento**
- O framework sincroniza tudo automaticamente

Isso resulta em:
- Código mais limpo
- Menos repetição
- Melhor legibilidade
- Manutenção mais simples

---

## ✅ Exemplo prático no projeto

A lista de personagens é gerada automaticamente a partir de um array JavaScript (`characters`).  
Qualquer alteração nesse array reflete imediatamente na interface, sem código adicional de atualização do DOM.
